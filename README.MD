## Install
```
npm i
```

## Run dev server
Runs a browsersync dev server, served from the *dist* folder.
```
npm run start
```

## Recompile dev files
Recompiles the *dist* folder without browsersync.
```
npm run compile
```

## Ready for poduction
Run when ready for production. Generates a *dist* folder.
```
npm run prod
```
