## Install
```
npm i
```

## Run dev server
Runs a browsersync dev server, served from *build*.
```
npm run start
```

## Recompile (./build/)
Recompiles the *build* folder without drowsersync.
```
npm run compile
```

## Poduction (./dist/)
When ready for production. Generates a *dist* folder.
```
npm run prod
```
